# CipherBox (CryptX)


A small Python CLI tool for encrypting and decrypting data using AES, Base64, and XOR.

## Fitur Utama

CLI interaktif & mudah digunakan

AES dengan password

Base64 (encode/decode)

XOR cipher dengan custom key

Sudah mendukung entrypoint CLI: cipherbox
